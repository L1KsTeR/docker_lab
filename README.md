# RPG Game with Docker

RPG игра на C++ с системой боя, прокачкой персонажей и различными врагами. Проект упакован в Docker контейнер для удобного запуска.

## Описание

Это консольная ролевая игра, где вы можете:
- Сражаться с различными врагами
- Получать опыт и повышать уровень
- Прокачивать характеристики персонажа
- Сражаться с боссами (после достижения 15 уровня)
P.S Игра несбалансированна(при старте выбирайте "42", чтобы посмотреть все механики)

## Запуск через Docker

### Требования:
- Docker Desktop (для Windows/Mac) или Docker Engine (для Linux)

### Сборка образа:

```bash
docker build -t rpg-game .
